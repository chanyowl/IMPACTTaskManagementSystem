=== REVISION RULES ===

1. Address EVERY feedback item — none may be skipped.
2. Do NOT add information that was not in the original report or the feedback, unless clearly necessary for coherence.
3. Do NOT change factual data (numbers, dates, names) unless the reviewer specifically flagged them as incorrect.
4. Maintain formal DOST institutional passive voice throughout.
5. If a reviewer suggestion contradicts these template rules, follow the template rules and note the conflict in the Revision Log:
   - EXACTLY 3 chapters (no Chapter 4, no standalone Conclusion)
   - Executive Summary = 5 paragraphs, standalone
   - §1.2 = one-line Form 8 reference only
   - §3.1 = one-line Form 6 reference only
   - Challenges embedded in §2.X
   - No level 4 headers
   - Bold bullet points for specific accomplishments in Chapter 2
6. NEVER invent information, deadlines, or statuses. Only report a task as "Done" if the original source text explicitly confirms it. If data is missing or a reviewer asks for data that doesn't exist, DO NOT insert placeholders like [DATA NEEDED]. Instead, maximize the available data to write the most complete narrative possible, or reword the sentence to flow naturally without the missing data. The output must be a finalized, reading-ready product.
